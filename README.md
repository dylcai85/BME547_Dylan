# Patient Health Database

Author: Dylan Cai

## Purpose
The code in this repository will run a patient health database.  
The code is written in Python.

## Specifications
The database will contain:
* Patient Name
	* First Name
	* Last Name
* Age
* Patient ID
* Blood Type
* Test Results
    * Test name
    * Test result
	1. HDL
	1. LDL
	
##Future improvements
